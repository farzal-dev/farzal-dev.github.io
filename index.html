About

I design and ship production-grade distributed systems — the kind that stay calm under load and cheap to run. Most of my work lives in the layer users never see: API gateways, message queues, caching strategies, and the serverless plumbing that keeps latency low and bills predictable.

Across the last nine years I've migrated monoliths to serverless AWS Lambda, built event-driven pipelines with RabbitMQ & SQS, cut database load with smart Redis caching, and integrated payment systems handling real money at scale. I care about clean architecture, measurable performance wins, and code other engineers actually enjoy maintaining.

Lately I've been exploring AI/ML integration in backend systems — bringing inference and intelligent automation into the services I build.
